blastoise syntax
================

blastoise syntax is inspired in golang and python syntax.

functions
---------

	func add(x, y) {
		return x + y
	}


	func sub(x, y) {
		return x - y
	}

loops
-----

	for item in [1, 2, 3] {
		print item
	}

	text = ""
	for text != "end" {
		text = read()
		print text
	}
